# 错题本完整流程

## 前置条件
- 需要切换到学校角色（admin 默认是 platform_admin，需要 switch-role）

## 步骤

### 1. 登录并切换角色
- 登录 admin/123456
- 如果当前角色不是学校管理员，点击角色切换切到学校角色
- 验证: 侧边栏出现"错题本"菜单

### 2. 进入错题本
- 点击侧边栏"错题本"
- 验证: 页面标题包含"错题"相关文字
- 验证: 数据表格/列表可见
- 截图: error-book-list

### 3. 数据验证
- 验证: 列表中有数据行（不是空状态）
- 验证: 每行包含题目信息（题号/科目/得分等）
- 验证: 有掌握状态筛选器（未掌握/练习中/已掌握）
- 截图: error-book-data

### 4. 查看错题详情
- 点击列表中第一条错题记录
- 验证: 弹出详情面板或跳转到详情页
- 验证: 详情中包含题目内容
- 验证: 详情中包含 AI 反馈（如果有）
- 截图: error-book-detail

### 5. 筛选功能
- 选择掌握状态筛选为"未掌握"
- 验证: 列表数据更新（数量变化或内容变化）
- 截图: error-book-filtered
