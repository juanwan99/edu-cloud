# 知识骨架可视化 设计方案

## 核心定位

**知识骨架是客观的、不因学生而异的知识结构。** 来自课标、教材、高考题，回答：
- 本省高考要求掌握**哪些知识和能力**
- 掌握到**什么程度**
- 知识之间**什么关系**

不叠加任何学情数据。学情是上层消费产品，不是骨架本身。

### 增量 vs 新建论证

**默认立场：增强已有。** 现有 17 个前端组件 + 9 个后端端点已覆盖完整的数据链路。

**问题不是缺功能，而是呈现方式不符合人的认知**——把后端图谱数据（nodes+edges）直接用力导向图渲染，等于展示数据库 schema。

**改造策略**：
- 保留全部后端端点和数据层（不新建）
- 新增 2-3 个后端预聚合端点（module-map / study-unit-detail / exam-patterns），消费已有 service
- 重构前端呈现层：将 ConceptMapPanel（G6 图）降级为审核工具入口，新增课程地图视图为教师默认入口
- 复用已有组件：NodeDetailDrawer/ExamItemsTab/StudyUnitTab/TreeNavPanel 可直接在新视图中使用

## 设计理念

### 核心隐喻：课程地图，不是知识图谱

教师看到的是一张**可教学、可备课、可查证的地图**：
- 5 个模块是**区域**
- 99 个学习单元是**路线站点**
- 108 个 L1 概念是**承重节点**
- 课标是**验收标准**
- 高考真题是**压力测试**

### 认知路径（教师打开页面的信息获取顺序）

1. **全局总览** → 高中生物高考由哪 5 块组成，哪块是基础，哪块是迁移出口
2. **选择模块** → 这个模块的教学主线、核心概念簇、学习先后顺序
3. **进入单元** → 这节课要讲什么，前面依赖什么，后面支撑什么
4. **查看概念** → 课标怎么要求，教材在哪里讲，高考怎么考，为什么这样排序
5. **展开证据** → 课标原文、教材页、真题、关系 evidence（默认不铺满）

### 布局原则

- 首屏：5 张模块卡 + 路标 + 右侧摘要 — 信息密度适中
- 模块页：左概念簇 + 中学习路线 + 右课标/高考侧栏
- 单元页：备课工作台风格
- 概念页：右侧抽屉或全屏详情
- **关系不画线**，改为标签、路径卡、对照表

### 信息密度控制

- 首屏不超过 5 张模块卡 + 6-8 条跨模块路标
- 模块页每屏 7±2 个重点概念，超出折叠
- 真题不列 2448 道，展示"考法画像 + 代表题"
- 原始 ID、q_matrix、edge type 默认不出现

## 高考真题和课标呈现

### 真题关联链路

`概念 → diagnostic_attributes → q_matrix → assessment_items`

前端不说 Q-matrix，翻译为：
- 这个概念被哪些**能力表现**考过
- 这些能力表现在哪些**高考题**中出现
- 题目要求学生把概念迁移到**什么距离**

### transfer_band 翻译

| 原始值 | 教师语言 | 含义 | 复习策略 |
|--------|---------|------|---------|
| near | 基础调用 | 直接调用概念、定义、结构、过程 | 先保底 |
| mid | 情境应用 | 在常规实验、图表、材料中使用 | 再训练 |
| far | 综合迁移 | 跨模块、陌生情境、多步骤推理 | 最后配置 |

**不是简单"难度"，而是概念到题目情境的迁移距离。**

### 课标要求呈现

- content → 显示为"内容要求"
- academic → 显示为"学业能力要求"
- 动词提取为掌握程度标签：`说出 / 举例 / 概述 / 阐明 / 解释 / 建构 / 探究`
- 原文默认折叠，外层只显示掌握程度标签 + 一句话摘要

## 概念关系的人性化表达

### 四类教学语言

| 原始 relation_type | 教师语言 | 含义 |
|-------------------|---------|------|
| prerequisite_hard | **必经前置** | 没掌握 A，学习 B 会卡住 |
| prerequisite_soft | **建议铺垫** | 先理解 A，学习 B 更顺 |
| bridge_to | **跨模块桥** | A 会在其他模块中再次使用 |
| contrast | **易混对照** | A 与 B 容易混，需要辨析 |

### evidence 融入

不作为字段展示，改成关系卡里的"**为什么**"：

```
细胞学说 → 原核/真核细胞
为什么：先建立"细胞是生命基本单位"的概念，才能对细胞进行原核/真核分类。
```
