## 0. 设计决策

| 决策 | 结论 | 理由 |
|------|------|------|
| 作业类型 | 3 类型起步（regular/pre_exam/post_exam）| haofenshu 5 类型简化；task_type 字符串枚举，后续加类型零改表 |
| 模块位置 | 独立 `modules/homework/`，与 exam 平行 | 1 模块 = 1 目录惯例；通过 exam_id FK 关联考试，不耦合代码 |
| 批改模式 | 仅手动批改（教师打分+评语）| 扫描 OCR/AI 自动批改留 grading_mode 字段扩展 |
| 前端 | 不含 | 后端先行，前端后续 Phase 单独做 |
| AI 推荐 | recommend_remedial 为 stub | Phase 3 学情分析完成后实现 |

## 1. 数据模型

### 1.1 homework_tasks 表（作业任务）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | String(36) | PK, UUID | IdMixin |
| school_id | String(36) | FK→schools, NOT NULL | TenantMixin |
| title | String(200) | NOT NULL | 作业标题 |
| task_type | String(20) | NOT NULL, default="regular" | regular / pre_exam / post_exam |
| subject_code | String(20) | NOT NULL | 科目代码 |
| class_id | String(36) | FK→classes, nullable | 目标班级（nullable 允许多班级扩展） |
| assigned_by | String(36) | FK→users, NOT NULL | 布置教师 |
| exam_id | String(36) | FK→exams, nullable | 关联考试（post_exam 必填） |
| deadline | DateTime | nullable | 截止时间 |
| status | String(20) | NOT NULL, default="draft" | 状态机 |
| content | Text | nullable | JSON：题目/描述/附件引用 |
| grading_mode | String(20) | default="manual" | 扩展点：manual / auto / scan |
| created_at | DateTime | TimestampMixin | 自动 |
| updated_at | DateTime | TimestampMixin | 自动 |

**状态机**：

```
draft ──→ active ──→ expired ──→ closed
  │                    │
  └──→ closed          └──→ closed
```

- `draft → active`：教师发布（自动创建 submissions）
- `active → expired`：超过 deadline（应用层检查，非自动）
- `active/expired → closed`：教师手动关闭或全部批改完成
- `draft → closed`：取消草稿

**索引**：`(school_id, status)`, `(school_id, class_id)`, `(assigned_by)`

### 1.2 homework_submissions 表（提交记录）

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | String(36) | PK, UUID | IdMixin |
| task_id | String(36) | FK→homework_tasks, NOT NULL | 所属作业 |
| student_id | String(36) | FK→students, NOT NULL | 提交学生 |
| status | String(20) | NOT NULL, default="pending" | pending / submitted / graded |
| score | Float | nullable | 得分 |
| feedback | Text | nullable | 教师评语 |
| submit_time | DateTime | nullable | 提交时间 |
| content | Text | nullable | JSON：答案内容 |
| graded_by | String(36) | FK→users, nullable | 批改人 |
| graded_at | DateTime | nullable | 批改时间 |
| created_at | DateTime | TimestampMixin | 自动 |
| updated_at | DateTime | TimestampMixin | 自动 |

**唯一约束**：`(task_id, student_id)`
**索引**：`(task_id, status)`

### 1.3 权限枚举新增

```python
MANAGE_HOMEWORK = "manage_homework"   # 布置/编辑/关闭/批改
VIEW_HOMEWORK = "view_homework"       # 查看作业/提交/统计
```

RBAC 映射：
- `manage_homework`：platform_admin, principal, academic_director, homeroom_teacher, subject_teacher
- `view_homework`：上述 + grade_leader, parent

### 1.4 模块注册

`MODULE_CODES` 新增 `"homework"`，`DEFAULT_ENABLED` 包含 homework。

### 1.5 Capability 映射

沿用现有 `read/write` 二元模型（与 `CAPABILITY_ACTIONS` 一致）：

| Capability | 用途 |
|------------|------|
| `homework.read` | 查看作业列表/提交/统计 |
| `homework.write` | 创建/发布/批改（合并 assign+grade） |

## 2. Service 层

### 2.1 HomeworkTaskService

| 方法 | 参数 | 逻辑 | 返回 |
|------|------|------|------|
| `create_task` | db, *, school_id, title, task_type, subject_code, class_id, assigned_by, exam_id, deadline, content | 校验：post_exam 必须有 exam_id；exam_id 存在性检查；assigned_by 归属校验 | HomeworkTask |
| `list_tasks` | db, *, school_id, class_id?, subject_code?, status?, task_type?, assigned_by? | 链式过滤，按 created_at DESC | list[HomeworkTask] |
| `get_task` | db, *, task_id, school_id | 404 校验 | HomeworkTask |
| `update_task` | db, *, task_id, school_id, **fields | 仅 draft 状态可编辑 title/content/deadline/class_id | HomeworkTask |
| `transition_status` | db, *, task_id, school_id, action | 状态机转换 | HomeworkTask |
| `delete_task` | db, *, task_id, school_id | 仅 draft 可删（硬删除） | None |

**状态转换规则**：

```python
TRANSITIONS = {
    "publish": {"from": ["draft"], "to": "active"},
    "expire":  {"from": ["active"], "to": "expired"},
    "close":   {"from": ["active", "expired", "draft"], "to": "closed"},
}
```

非法转换抛 `StateTransitionError`（复用 2.1 模式）。

**publish 副作用**：`transition_status("publish")` 内部自动调用 `HomeworkSubmissionService.create_submissions_batch()`，查询 class_id 对应的全部学生，批量创建 pending 记录。**publish 前置条件**：class_id 不能为空（nullable 仅为未来多班级扩展预留），否则抛 ValidationError。

### 2.2 HomeworkSubmissionService

| 方法 | 参数 | 逻辑 | 返回 |
|------|------|------|------|
| `create_submissions_batch` | db, *, task_id, student_ids | 批量创建 pending 记录（幂等，已存在的跳过） | int (created count) |
| `submit` | db, *, submission_id, content | pending → submitted，记录 submit_time | HomeworkSubmission |
| `grade_single` | db, *, submission_id, score, feedback, graded_by | submitted → graded，记录分数/评语/批改人/时间 | HomeworkSubmission |
| `grade_batch` | db, *, task_id, grades: list[{student_id, score, feedback}], graded_by | 批量打分，逐条校验，跳过无效 student_id | int (graded count) |
| `list_submissions` | db, *, task_id, status? | 按 student_id 排序 | list[HomeworkSubmission] |
| `get_task_stats` | db, *, task_id | 统计：total/submitted/graded/pending + submission_rate + avg_score | dict |

### 2.3 @audited 集成

`create_task`、`transition_status`、`grade_batch` 三个写操作加 `@audited` 装饰器。

## 3. API 端点

前缀 `/api/v1/homework`，全部 JWT 认证。

### 3.1 作业管理

| 方法 | 路径 | 权限 | 用途 |
|------|------|------|------|
| POST | `/tasks` | MANAGE_HOMEWORK | 创建作业（draft） |
| GET | `/tasks` | VIEW_HOMEWORK | 列表（支持 class_id/subject_code/status/task_type/assigned_by 过滤） |
| GET | `/tasks/{id}` | VIEW_HOMEWORK | 详情（含统计摘要） |
| PATCH | `/tasks/{id}` | MANAGE_HOMEWORK | 更新（仅 draft） |
| POST | `/tasks/{id}/publish` | MANAGE_HOMEWORK | 发布（draft→active，自动创建 submissions） |
| POST | `/tasks/{id}/close` | MANAGE_HOMEWORK | 关闭 |
| DELETE | `/tasks/{id}` | MANAGE_HOMEWORK | 删除（仅 draft） |

### 3.2 提交与批改

| 方法 | 路径 | 权限 | 用途 |
|------|------|------|------|
| GET | `/tasks/{id}/submissions` | VIEW_HOMEWORK | 提交列表（支持 status 过滤） |
| POST | `/tasks/{id}/submissions/{sid}/submit` | VIEW_HOMEWORK | 学生提交 |
| POST | `/tasks/{id}/submissions/{sid}/grade` | MANAGE_HOMEWORK | 单条批改 |
| POST | `/tasks/{id}/grade-batch` | MANAGE_HOMEWORK | 批量批改 |
| GET | `/tasks/{id}/stats` | VIEW_HOMEWORK | 统计 |

### 3.3 数据范围（ScopeFilter）

| 角色 | 可见范围 |
|------|---------|
| platform_admin / principal | 全校 |
| academic_director | 全校 |
| grade_leader | 本年级班级 |
| homeroom_teacher | 本班全部 |
| subject_teacher | 自己布置的 + 所教班级所教科目的 |
| parent | 仅自己孩子的提交 |

注册在 `ai/tools/homework.py`，`module_code="homework"`, `domain="homework"`。

| 工具名 | category | risk | 描述 | requires_capabilities |
|--------|----------|------|------|----------------------|
| `list_homework_tasks` | L2_homework | low | 列出作业（支持 class/subject/status 过滤） | homework.read |

| `get_submission_details` | L2_homework | low | 查看某作业的提交明细 | homework.read |

| `recommend_remedial` | L2_homework | low | **stub** — 考后补偿推荐（返回"功能开发中"） | homework.read |

### Agent 触发规则（Phase 1e 实现，本 Phase 仅声明）

| 事件 | 条件 | 输出 | 接收角色 |
|------|------|------|---------|
| 提交率低 | active 作业 submission_rate < 50% 且距 deadline < 24h | 催交提醒 | assigned_by |
| 全部批改完成 | graded_count == total_count | 建议关闭作业 | assigned_by |

## 6. 测试策略

| 层 | 测试文件 | 预估用例数 | 覆盖 |
|----|---------|-----------|------|
| Service | test_homework_service.py | ~25 | CRUD + 状态转换（合法+非法）+ 批量创建 + 批改 + 统计 + post_exam 校验 |
| API | test_homework_api.py | ~20 | 端点 CRUD + 权限拒绝 + ScopeFilter + 状态转换 API |
| AI Tools | test_homework_tools.py | ~8 | 5 工具基本调用 + 权限注入 + stub 返回 |
| Migration | test_alembic_migration.py | 已有 | 更新期望表集合（+2 表） |

**关键测试场景**：
- post_exam 作业创建时 exam_id 为空 → ValidationError
- draft 状态发布 → submissions 自动创建（数量 = 班级学生数）
- 非 draft 状态尝试编辑 → StateTransitionError
- subject_teacher 只能看自己布置的作业
- grade_batch 部分 student_id 不存在 → 跳过无效 + 返回成功数
- 已 graded 的 submission 重复批改 → ConflictError
- active 作业关闭后不接受新提交

预估新增 **~53 tests**。
